About Guane Overview

A multilingual, user-friendly Shiny platform for phylogenetic comparative analyses

Guane

Guane is a Shiny web platform aggregating the full breadth of phylogenetic comparative methods (PCMs) into a single, graphically-driven environment. Named after the pre-Hispanic Guane people of northeastern Colombia, the word also means "tree" in the Chibcha language — a perfect echo of the phylogenies at its core.

Built to remove language, programming, and access barriers from rigorous evolutionary analysis. Guane integrates established R libraries into a unified GUI with a Live Code Mirror, multilingual interface, and integrated chatbot support.

💡 Core Innovation

Guane is the first platform to simultaneously provide validated, multilingual chatbot access to the full range of PCM methods, reinforce computational reproducibility, and integrate pre-registration schemes.

The Live Code Mirror — a reactive, editable R script panel — exports imperative code ready for any command-line environment. Not implemented in any existing PCM software.

Every analysis produces publication-quality figures, an exportable R script, and a Quarto document combining embedded code, results, and narrative summary.

Analytical Modules 4 Core Modules

Independent modules covering the main phylogenetic comparative analyses in R

🌿
Module 01

Phylogenetic Regression & Comparative Linear Models

Blomberg's K and Pagel's λ for phylogenetic signal. Phylogenetic Independent Contrasts (PICs). PGLS with BM, Grafen, Pagel, and Blomberg correlation structures. PGLM for binary, count, and proportion outcomes. AIC model comparison and ranking.

phytools · ape · geiger · nlme · brms
🌿
Module 02

Ancestral State Reconstruction

Continuous characters under BM, Early Burst, and Ornstein–Uhlenbeck via ML. Discrete characters (Mk, ARD, SYM) via ML and Bayesian MCMC. Marginal, joint, and stochastic mapping reconstructions. Pagel's test for co-evolution. Threshold model support.

phytools · ape · diversitree
🌿
Module 03

Diversification Rate Analysis

Yule, birth–death, time-variable, clade-dependent, and diversity-dependent models. Gamma statistic with Monte Carlo CRTT. AIC model comparison and model averaging. MCMC posterior estimation for complex models. Convergence via ESS and Geweke test.

ape · diversitree · DDD · TESS
🌿
Module 04

Character-Dependent Diversification

BiSSE, MuSSE, QuaSSE, and HiSSE models under ML and Bayesian MCMC. Full AIC comparison tables per model class. Posterior density plots per parameter. Chain convergence diagnostics via ESS and Geweke test using the TESS engine.

diversitree · hisse · TESS

🪞 Live Code Mirror

The central innovation of Guane. Every GUI interaction generates a reactive, editable R script in a shinyAce panel. Though Shiny follows a declarative style, exported code has an imperative structure executable on any server, IDLE, or notebook.

Follows Google's R Style Guide with full inline comments. Researchers can export to:

R script (.R) Quarto (.qmd) SVG / PNG RStudio-ready
Reproducibility

Workflows as versioned, auditable DAGs. Docker + renv guarantee long-term reproducibility.

Accessibility Architecture Section 3.2.5

Breaking linguistic, technical, and learning barriers in phylogenetic methods

🌍 Multilingual Interface

Full trilingual interface using Shiny.i18n, covering documentation, error messages, parameters, diagnostics, and results.

Dictionaries are structured JSON tables, available on GitHub. Language switching is reactive within a running session.

Build Setup & Install

Deploy Guane locally or on a server via Docker or direct R installation

🐳 Docker (Recommended)

Containerized deployment with locked library versions via renv.

git clone https://github.com/alarconvv/Guane
cd Guane
docker build -t guane .
docker run -p 3838:3838 guane
📦 R Direct Install

Run locally using R ≥ 4.3 and renv for reproducible environments.

# In R console
install.packages("renv")
renv::restore()
shiny::runApp("app.R")
